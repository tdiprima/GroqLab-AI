## Getting Started

To run the Python scripts in this repository, follow these steps:

1. **Set the `GROQ_API_KEY` environment variable**:

    ```bash
    export GROQ_API_KEY="your_groq_api_key"
    ```

2. **Run the Python scripts**:

    ```bash
    python path/to/script.py
    ```
